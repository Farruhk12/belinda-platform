# Belinda Platform

Единый веб-портал — центральная точка доступа ко всем инструментам компании Belinda.

## Запуск

```bash
npm install
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Сборка

```bash
npm run build
npm start
```

## Функции

- **Категории инструментов** — Аналитика, Работа МП, Обучение, Внутренние сервисы, Автоматизация
- **Поиск** — по названию, описанию и категории
- **Избранное** — закрепление часто используемых инструментов (localStorage)
- **Профиль** — мини-профиль пользователя
- **Счётчик пользователей** — отображает количество онлайн
- **Новые инструменты** — отдельный блок для недавно добавленных
- **Уведомления** — о обновлениях систем

## Технологии

- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript

## Структура

- `src/data/tools.ts` — данные категорий и инструментов
- `src/components/` — компоненты интерфейса
- `src/app/page.tsx` — главная страница
